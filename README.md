#  Tour Site Website

A modern and responsive tourism website built using HTML, CSS, JavaScript, Tailwind CSS, DaisyUI, and AOS Animations. The website showcases Pakistan's beautiful destinations, upcoming tours, travel packages, and contact information.

---

##  Features

- Responsive Navigation Bar
- Mobile Menu Toggle
- Tab-Based Navigation System
- Hero Section
- About Us Section
- Destination Gallery
- Upcoming Tours Section
- Signature Tours
- Pakistan Destinations Showcase
- Contact Form
- Smooth Scroll Animations (AOS)
- Mobile & Tablet Responsive Design

---

##  Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- DaisyUI
- Font Awesome
- AOS (Animate On Scroll)

##  Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/tour-site.git
```

2. Open the project folder

```bash
cd tour-site
```

3. Run the project

Simply open:

```bash
home.html
```

in your browser.
